## 🔨 What I'm Building

### [Job Application Tracker](https://github.com/MillerA5051/app-tracker) &nbsp;•&nbsp; `In Progress`
A full-stack web app for tracking job applications end-to-end — statuses, contacts, notes, cover letters, and a live metrics dashboard.

**Stack:** Python · FastAPI · PostgreSQL · SQLAlchemy · React · JWT Auth · Email 2FA

- Normalized relational schema across 4 tables (applications, contacts, users, verification codes)
- Secure auth system with JWT tokens + email-delivered 6-digit verification codes
- Real-time dashboard surfacing total applications, interviews planned, offers received, and response rate
- Deploying to Netlify
